# My React Blog Site

This blogging site was created using Ruby on Rails and JavaScript's React as the final portfolio project for Flatiron School's Online Full-stack Web Developer Curriculum. It is a single page application that allows one to create, delete, and view blogposts. The site is developed with separate API and client front-end.

## Installation and Usage

Clone this repository.
Run `cd my-react-blog-site-api` to change into the back-end directory.
Run `rake db:migrate` to establish the database.
Run `rails server -p 3001` to host the api on a unique port.
Create a new tab in your command line.
Run `cd ../my-react-blog-site-client` to change into the front-end directory.
Run `npm install` to install packages.
Rum `npm start` to host the client app on a unique port.
In your browser, go to localhost:3000 to use the blog app!
